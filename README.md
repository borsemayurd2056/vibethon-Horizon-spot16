# 🚀 AIML PlayLab

An interactive Artificial Intelligence & Machine Learning learning platform designed to make learning fun, visual, and engaging through quizzes, games, and simulations.

---

## 🎯 Project Objective

AIML PlayLab is built to simplify complex AI/ML concepts using:
- Interactive quizzes
- Educational mini-games
- Real-world ML simulations
- Optional authentication system

The goal is to help users **learn AI/ML by doing, not just reading**.

---

## ✨ Features

### 🧠 Quiz System
- Level-based quiz (Beginner / Intermediate / Advanced)
- Instant scoring system
- Detailed answer review after submission
- Explanations for each question
- Performance analysis with learning suggestions

---

### 🎮 AI Learning Games

#### 1. Scenario-Based Game
- Real-world AI decision-making scenarios
- Improves logical thinking

#### 2. KNN Classifier Game
- Demonstrates how machine learning predicts using nearest data points
- Based on K-Nearest Neighbors algorithm

#### 3. Gradient Descent Racer (Optional/Bonus)
- Visual simulation of optimization process
- Teaches how ML models minimize loss

---

### 🔐 Authentication (Optional)
- Supabase OAuth integration
- Login with:
  - Google
  - GitHub
- Guest access supported (no login required for basic features)

---

### 📊 Dashboard
- User progress tracking
- Quiz score summary
- Learning suggestions
